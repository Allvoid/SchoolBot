Квадратичная функция
Уравнение: y = ax² + bx + c.

График — парабола.

Вершина параболы:
x₀ = -b/(2a),
y₀ = a(x₀)² + b(x₀) + c.

Направление ветвей:

Если a > 0, парабола открыта вверх.

Если a < 0, парабола открыта вниз.

Нули функции: ax² + bx + c = 0 (решение квадратного уравнения).

Степени и их свойства
aᵐ ⋅ aⁿ = aᵐ⁺ⁿ (произведение степеней с одинаковым основанием).

aᵐ / aⁿ = aᵐ⁻ⁿ (частное степеней с одинаковым основанием, a ≠ 0).

(aᵐ)ⁿ = aᵐⁿ (степень степени).

(ab)ⁿ = aⁿ ⋅ bⁿ (степень произведения).

(a/b)ⁿ = aⁿ / bⁿ (степень дроби, b ≠ 0).

a⁰ = 1 (при a ≠ 0).

a⁻ⁿ = 1/aⁿ (отрицательная степень, a ≠ 0).

Рациональные выражения
Упрощение дробей: (ac)/(bc) = a/b (сокращение на общий множитель, c ≠ 0).

Сложение и вычитание: привести к общему знаменателю, например:
a/b + c/d = (ad + cb)/(bd) (при b ≠ 0, d ≠ 0).

Умножение: (a/b) ⋅ (c/d) = (ac)/(bd).

Деление: (a/b) ÷ (c/d) = (a/b) ⋅ (d/c) = (ad)/(bc) (при c ≠ 0).

Разложение многочленов на множители
Вынесение общего множителя: ax + bx = x(a + b).

Группировка:
ax + bx + ay + by = x(a + b) + y(a + b) = (x + y)(a + b).

Использование формул сокращённого умножения:
x² + 2x + 1 = (x + 1)².
