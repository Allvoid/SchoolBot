/ - деление 
* - умножение
% - процент
Сложение и вычитание:

a + b = c

a - b = c

Умножение и деление:

a * b = c

a / b = c

Порядок действий:

Скобки, умножение/деление, сложение/вычитание.

Дроби:

a/b + c/d = (ad + bc)/bd

a/b * c/d = ac/bd

Проценты:

Процент = (Часть / Целое) * 100%

Площадь и периметр:

Площадь прямоугольника: S = a * b

Периметр прямоугольника: P = 2(a + b)

Площадь квадрата: S = a²

Периметр квадрата: P = 4a

Площадь треугольника: S = (1/2) * основание * высота

Периметр треугольника: P = a + b + c

Окружность и круг:

Длина окружности: C = 2πr

Площадь круга: S = πr²
