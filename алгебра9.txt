Корни и радикалы

Квадратный корень:
√a ⋅ √b = √(ab) (при a ≥ 0, b ≥ 0).
√(a/b) = √a / √b (при a ≥ 0, b > 0).
√(a²) = |a|.

Степень с рациональным показателем:
a^(m/n) = ⁿ√(aᵐ) (при a > 0, если n четное).

Приведение подобных радикалов:
2√3 + 5√3 = 7√3.

Арифметическая прогрессия

n-й член:
aₙ = a₁ + (n - 1)d, где d — разность, a₁ — первый член.

Сумма n членов:
Sₙ = (n/2)(a₁ + aₙ) или Sₙ = (n/2)[2a₁ + (n - 1)d].

Геометрическая прогрессия

n-й член:
bₙ = b₁ ⋅ q^(n - 1), где q — знаменатель, b₁ — первый член.

Сумма n членов:
Sₙ = b₁ ⋅ (qⁿ - 1)/(q - 1) (при q ≠ 1).

Сумма бесконечной геометрической прогрессии (при |q| < 1):
S = b₁ / (1 - q).

Квадратные неравенства

Решение ax² + bx + c > 0 (или < 0):

Найти корни x₁ и x₂ (x₁ < x₂).

Определить знаки на интервалах (-∞, x₁), (x₁, x₂), (x₂, +∞) с учетом a.

Если a > 0: > 0 вне корней, < 0 между корнями.

Если a < 0: наоборот.

Подстановка: x² = kx + b, затем решение получившегося квадратного уравнения.

Арифметическая прогрессия:

aₙ = a₁ + d(n-1)

Сумма: Sₙ = n(a₁ + aₙ) / 2

Геометрическая прогрессия:

bₙ = b₁ * q^(n-1)

Сумма: Sₙ = b₁ * (qⁿ - 1) / (q - 1)

Логарифмы:

logₐ(b * c) = logₐb + logₐc

logₐ(b / c) = logₐb - logₐc

logₐ(b^c) = c * logₐb
