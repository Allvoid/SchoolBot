Тригонометрия в прямоугольном треугольнике

Синус: sin α = a / c, где a — противолежащий катет, c — гипотенуза.

Косинус: cos α = b / c, где b — прилежащий катет, c — гипотенуза.

Тангенс: tg α = a / b = sin α / cos α.

Котангенс: ctg α = b / a = cos α / sin α.

Основное тригонометрическое тождество: sin² α + cos² α = 1.

Теорема косинусов

Для любого треугольника:
a² = b² + c² - 2bc ⋅ cos α,
где a, b, c — стороны, α — угол напротив стороны a.

Теорема синусов

Для любого треугольника:
a / sin α = b / sin β = c / sin γ = 2R,
где R — радиус описанной окружности.

Вписанная и описанная окружности

Радиус вписанной окружности треугольника:
r = S / p,
где S — площадь, p = (a + b + c) / 2 — полупериметр.

Радиус описанной окружности треугольника:
R = a / (2 sin α).

Стереометрия (начала)

Объем прямоугольного параллелепипеда:
V = a ⋅ b ⋅ c,
где a, b, c — длина, ширина, высота.

Площадь полной поверхности прямоугольного параллелепипеда:
S = 2(ab + bc + ac).

Объем куба:
V = a³,
где a — ребро.

Площадь полной поверхности куба:
S = 6a².

Длина диагонали прямоугольного параллелепипеда:
d = √(a² + b² + c²).

Объем цилиндра:
V = πr²h,
где r — радиус основания, h — высота.

Площадь боковой поверхности цилиндра:
S_бок = 2πrh.

Площадь полной поверхности цилиндра:
S_полн = 2πr(r + h).
